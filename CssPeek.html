Shift + Ctrl -> "judul" Melihat Kode CSS dengan Mudah: CSS Peek memungkinkan Anda melihat kode CSS yang terhubung dengan elemen HTML atau ejs yang sedang Anda kerjakan. Anda dapat menyorot nama class atau ID, dan ekstensi ini akan menampilkan definisi CSS yang relevan. Navigasi Cepat ke Definisi: Anda dapat langsung menuju ke file CSS atau membukanya dalam editor baru dengan mengklik nama class atau ID (Ctrl+Shift). Selain itu, Anda dapat melihat definisi symbol dengan mengarahkan kursor dan menekan Ctrl saat menghover pada nama class atau ID.
